Get started in minutes

Install the Bundle

Ship the same stack across macOS, Linux, and Windows—no extra build steps required.

Step 1

Grab the package

Install the CLI, runtime, and labs from PyPI.

pip install thebundle

Step 2

Launch locally

Start the FastAPI-powered site and browse labs from the navbar.

bundle website start

Step 3

Explore the primitives

Inspect logging, telemetry, sockets, tracing, and browser utilities in one place.

python -m bundle --help

Why Bundle

Everything you need to wire modern tools together.

Bundle primitives share transports, observability, and process management so you can focus on your experiments.

Observability built in

Tracing, logging, and metrics are unified so labs and services emit the same signals.

Transport aware

Browser capable

Test focused

Advanced testing harnesses match the primitives, from BLE to YouTube integrations.
